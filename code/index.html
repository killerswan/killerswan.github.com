Iowa State University, winter

I have been hacking on software for ages, and I am always teaching myself new things. These are some recent examples, though my github will be more current. Enjoy!

Recent code

Triad
I'm experimenting with an app which uses the scrypt hash to generate more easily memorizable passwords in Python, including a Kivy GUI (for Android, Linux).

The name refers to a Sherlock Holmes episode which inspired this algorithm!

Rust
Rust is Mozilla's new concurrent programming language, and I've been contributing to Rust's standard libraries a lot, lately, and maintaining the Ubuntu PPA packages.

A couple of highlights you might like are my implementations of the Alioth k-nucleotide and chameneos-redux concurrency benchmarks, which you'll find here.

Askitter
I've been maintaining and contributing to my fork of these Haskell bindings to the Twitter API. Amongst other things, I've worked extensively with the following:
  1. twitter-upload, a program to upload and post images in tweets
  2. twitter-unfav, a program to remove old favorites from Twitter
  3. hoauth, a library used by Askitter for both OAuth and (via libcurl) SSL
Stanford CS240h labs
My version of each lab from the Mazieres and O'Sullivan course, CS240h: Functional Systems in Haskell (2011). Lab 2 uses a Hilbert R-Tree to store the coordinates of a set of rectangles and to spatially query them.
Haskell-BLAKE (hackage: blakesum, blakesum-demo)
An experimental Haskell implementation of the BLAKE SHA-3 candidate hash algorithms, including BLAKE-256, BLAKE-512, BLAKE-224, and BLAKE-384. (Based on the submitted BLAKE proposal.)
Ninlil, for Tumblr
An F# experiment in reading, reblogging, and deleting posts using Tumblr's API (v1)
Dragon curve in JavaScript
Fractal goodness in JavaScript/SVG! (Tested with Chrome.)
excelGrep
This is a tool for searching within Excel spreadsheets and then navigating to the cells with resulting matches: one thing I wish was better done by Excel 2003, at least. (Sorry, this does require IE and Microsoft Excel...) This is quite useful in only ~250 lines of JavaScript! (Note: may be broken by recent IE updates?)
