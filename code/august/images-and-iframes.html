File extension switching

13. The .htaccess file of a directory can be used to change the MIME type of a particular file extension, like so: 

# Add extension JPG to HTML mime type
AddType text/html .jpg

Thus, the following iframe shows a ".jpg" which is actually HTML (with another JavaScript alert):

14. Compare that file (extension-switcheroo/not.jpg) and another-not.jpg. The first will be served with a MIME type of "text/html", while the second will be served with a MIME type of "image/jpeg", because of the change made in the .htaccess file.

This can be confirmed, for example, by using wget in Cygwin.
