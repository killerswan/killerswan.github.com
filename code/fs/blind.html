F#, or the halt leading the blind!

In F# I see a lot of programming magic I want to use. There are a whole ton of the neat things from JavaScript and Underscore.js and CoffeeScript are there, plus a type system like Haskell's (without the explicit monads) in addition to things like units of measure, and a variety of strange animals like arrays, lists, and sequences: all watched over by the CLR / .NET parent alien.

I'm going to teach myself how to get started: maybe this will help you, too. I have to warn you, I'm not sure I've ever played any music in this key signature before...

A brief example from here:

let walkTree tree =
    let rec loop depth tree =
        let spacer = new string(' ', depth)
        match tree with
        | Leaf(value) ->       
            printfn "%s |- %A" spacer value
        | Node(tree1, tree2) ->
            printfn "%s |" spacer
            loop (depth + 1) tree1
            loop (depth + 1) tree2
    loop 0 tree
